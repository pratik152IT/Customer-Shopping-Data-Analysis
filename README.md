# Customer Shopping Data Analysis

## Overview
This project analyzes customer shopping behavior using a Kaggle retail dataset.
The goal is to identify sales trends, high-performing categories, and customer purchasing patterns.

## Tools Used
- Python
- Pandas
- Google Colab
- Tableau Public
- Kaggle Dataset

## Key Analysis
- Data cleaning and feature engineering
- Category-wise revenue analysis
- Monthly sales trend analysis
- Data visualization using Tableau
